# 在线支付订单

提交支付（本站） ---> 第三方支付平台  --> 银行站点 --> 第三方支付平台  --> 本站 

支付过程：
先POST发送给 第三方支付平台 信息， 如：订单号，总金额，商户号
返回过程：
第三方支付平台POST给本站 信息
如：支付结果，支付金额...
